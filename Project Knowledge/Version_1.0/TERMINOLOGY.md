## 1. Kernbegriffe Fensterbau

| Deutsch | Polnisch | Kontext / Anmerkung |
|---------|----------|---------------------|
| Fenster | Okno | Allgemein |
| Fensterbank (aussen) | Parapet okienny (zewnetrzny) | Material: Aluminium, Ausladung in cm |
| Fensterbank (innen) | Parapet okienny (wewnetrzny) | Seltener relevant |
| Fensterbankanschlussprofil (FB30) | Profil podparapetowy (FB30) | Verbindung Fenster-Fensterbank, Hoehe in mm |
| Fensterfluegel | Skrzydlo okienne | Beweglicher Teil |
| Fensterrahmen | Rama okienna | Fester Teil |
| Haustuer | Drzwi wejsciowe | Eingangsbereich |
| Innentuer | Drzwi wewnetrzne | Innenbereich |

## 9. Nutzungshinweise

### Fuer die Uebersetzungs-Engine:
- Diese Liste wird als System-Kontext bei jeder Uebersetzung mitgegeben
- Bei unbekannten Begriffen: Original beibehalten + in Klammern markieren
- Abkuerzungen (FB30, VSG, RC2) in beiden Sprachen gleich lassen
- Massangaben immer in mm (ausser explizit anders angegeben)
- Polnische Sonderzeichen korrekt verwenden (a-ogonek, l-barred, etc.)

### Erweiterung:
- Neue Begriffe aus jeder Korrespondenz extrahieren und hier ergaenzen
- E&W kann ueber Dashboard (Phase 5) eigene Begriffe hinzufuegen
- Pro Lieferant koennen Begriffsvarianten gespeichert werden
